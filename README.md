# Overview

As a software development student, I am continuing to build a personal software portfolio that demonstrates my growth in programming concepts and software development practices. This project is a TypeScript-based Adventure Experience Recommendation Toolkit that runs in the terminal using Node.js.

The software demonstrates sever core TypeScript concepts including:
- Classes
- Lists and arrays
- Functions
- Recursion
- Asynchronous functions
- Exception handling
- Terminal output

My program uses structured recommendation data to generate personalized adventure recommendations. It also includes a recursive search function, asynchronous data loading simulation, and exception handling validation logic.

The purpose of this project is to learn then strengthen my my understanding of TypeScript and learn how static typing improves code organization, maintainability, and error detection while building software that could later support applications.

# How to Run the Program

1. Download or clone the respoistory
2. Open the project folder in Visual Studio Code
3. Open a terminal in the project directory
4. Install dependencies: npm install
5. Run program: npm run dev
